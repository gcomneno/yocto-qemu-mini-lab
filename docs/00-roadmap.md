# Roadmap

## Phase 1 - Boot a minimal Yocto image

Build `core-image-minimal` and boot it with QEMU.

## Phase 2 - Add a custom layer

Create `meta-monkey`.

## Phase 3 - Add a custom recipe

Install a tiny `hello-monkey` command into the image.

## Phase 4 - Add a custom image

Create a minimal image that includes our package.

## Phase 5 - Use with vscode-bitbake

Open the workspace in VS Code and observe how the extension behaves on a real
Yocto project.
