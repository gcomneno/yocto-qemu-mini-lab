# Secure Boot radar

Source:
- Roy Jamil, Ac6
- Secure Boot for Embedded Linux: Explained in Simple Words
- Embedded Linux Conference NA 2026

Goal:
Understand the embedded Linux secure boot chain and map it later to a Yocto/QEMU learning lab.

Initial topics:
- Boot chain: ROM, bootloader, kernel, root filesystem.
- Secure boot vs verified boot vs measured boot.
- Signatures, hashes, keys, trust anchors.
- Yocto image signing possibilities.
- Possible future lab with QEMU and U-Boot.

Status:
Radar only. No implementation yet.
