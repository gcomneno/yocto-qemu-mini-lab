# yocto-qemu-mini-lab

A tiny educational Yocto/QEMU lab.

Goal:

- build a minimal Linux image with Yocto
- boot it with QEMU
- add a custom layer
- add a tiny custom package
- use the project as a real workspace for learning vscode-bitbake

This repository intentionally does not store Yocto build artifacts, downloads,
sstate cache, or cloned upstream sources.
